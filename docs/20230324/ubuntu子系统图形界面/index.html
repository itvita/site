安装图形界面

安装并配置远程桌面服务xrdp

安装xrdp

sudo apt install xrdp

配置xrdp端口（将远程端口改为3390，避免和本机的3389端口冲突）

sudo sed -i 's/port=3389/port=3390/g' /etc/xrdp/xrdp.ini

配置xsession

sudo echo xfce4-session >~/.xsession

重启xrdp服务

sudo service xrdp restart
