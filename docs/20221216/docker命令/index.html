说明：

  • –name nginx-test： 容器名称。
  • -p 8080:80： 端口进行映射，将本地 8080 端口映射到容器内部的 80 端口。
  • -d nginx： 设置容器在在后台一直运行。

查看当前运行中应用

说明

  • CONTAINER ID： 容器 ID
  • IMAGE： 使用的镜像
  • COMMAND： 启动容器时运行的命令
  • CREATED： 容器的创建时间。
  • STATUS： 容器状态
  • PORTS： 容器的端口信息和使用的连接类型（tcp\udp）。
  • NAMES： 自动分配的容器名称。
