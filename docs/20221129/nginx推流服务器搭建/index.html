推流

  1. 打开 OBS
  2. 添加场景
  3. 添加来源
  4. 开始推流
  5. 设置->推流->服务器设置为：rtmp://127.0.0.1:1935/live
  6. 登录监控地址http://127.0.0.1/stat 可以看到推流数据

拉流

  1. 打开VCL
  2. 媒体->打开网络串流->url设置为：rtmp://127.0.0.1:1935/live
